# 停开机服务

## 部署
### windows
1. 安装nodejs10.16  x86 或x64  for  windows 
2. 拷贝本项目压缩包到指定目录
3. 打开windows cmd
4. cd  指定目录
5. 执行 npm install 安装相关依赖
7. node nw-install.js 部署成服务
8. 检查是否安装成功
